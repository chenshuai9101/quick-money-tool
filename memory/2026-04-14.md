# 2026-04-14 记忆记录

## 上午工作：体感王者DIY项目开发

### 项目概述
成功创建了完整的"体感王者DIY"系统，这是一个用身体动作玩王者荣耀的低成本体感游戏方案。

### 已完成的核心组件
1. **姿势检测模块** (`pose_detector.py`) - 使用MediaPipe进行实时人体姿势检测
2. **动作映射引擎** (`action_mapper.py`) - 将姿势映射为游戏按键
3. **键盘模拟器** (`keyboard_emulator.py`) - 跨平台键盘输入模拟
4. **主应用程序** (`main_app.py`) - 整合所有组件的完整应用

### 技术挑战与解决方案
1. **摄像头权限问题**：macOS摄像头权限被拒绝，创建了多种解决方案：
   - 权限重置脚本 (`fix_camera_permission.sh`)
   - 模拟模式降级方案
   - 无摄像头测试版本

2. **MediaPipe兼容性问题**：MediaPipe 0.10.33版本API变化
   - 创建了新版兼容检测器 (`pose_detector_new.py`)
   - 实现了自动降级机制

3. **编码问题**：HTML文件中文乱码
   - 创建了多个编码修复版本
   - 确保UTF-8编码声明正确

### 用户界面开发
创建了多个HTML游戏界面：
1. **完整游戏界面** (`体感王者游戏.html`) - 专业游戏UI
2. **点击即玩界面** (`点击即玩体感王者.html`) - 简化快速版
3. **修复编码版本** (`体感王者游戏_最终版.html`) - 解决乱码问题

### 核心功能特点
- **实时姿势检测**：使用MediaPipe检测33个人体关节点
- **智能动作映射**：将身体动作映射为游戏按键
- **跨平台支持**：macOS/Linux/Windows兼容
- **自动降级**：摄像头失败自动切换到模拟模式
- **可配置性**：支持自定义手势和灵敏度设置

### 手势映射配置
默认配置包括：
- **移动控制**：身体倾斜 → WASD
- **技能释放**：手臂动作 → 1,2,3,4
- **特殊操作**：普攻(J)、回城(B)、闪现(F)
